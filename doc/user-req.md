Project name: Bravo My Life
Author : me

* 개요
- 브라보 마이 라이프는 사용자가 연도별로 자신의 삶을 검토하고 미래를 계획할 수 있는 웹 및 앱 애플리케이션입니다.
- 주로 시니어 연령대가 사용하며, 과거를 연도별로 돌아보는 기능에 촛점이 맞추어져 있다.

* 개발 환경
- Flutter 언어로 제작한다.
- windows, android, ios 를 지원한다.

* 기능
- 사용자가 자신의 인생 시작점을 입력하는 것으로 시작한다.
  예를 들어  1968년 이렇게 적으면, 이 때부터  현재까지, 그리고 미래의 100년을 연도별로 돌아볼 수 있다.
- 이것은 철저하게 년도별 다이어리라고 할 수 있다.  그래서 달력과 같은 그리드의 형태로 표시하되,  하나의 item, 즉 사각형 영역은 하나의 연도가 된다.
- 그리드의 가로 아이템 갯수는 디폴트로 12개여야만 한다.
- 따라서, 휴대폰에서 이 앱을 가동할 때는 가로모드로 구동되어야 하며, 필요시 스크롤바가 나와야 한다.
- 12개의 가로 아이템에서 마지막 3개 년도는 소위 "삼재" 에 해당하는 년도이므로, 별도의 구분선 컬러로 표시한다.
- 각 아이템은 이벤트를 입력할 수 있다. 이는 마치 구글 다이어리와 비슷하다고 생각할 수 있다.  다만 차이가 있다면, 하나의 사각형이 하루가 아닌 1년이라는 것이다.  즉 사용자는 해당 년도에 있었던 중요한 사건을 적어넣어 놓고 과거를 회상할 수 있다.
- windows 은 공간이 충분하므로, 대략적으로 이벤트를 표시할 수 있으나, app 은 공간이 없으므로, 이벤트 갯수만 표시하고, 클릭시, 이벤트의 내용을 팝업하여, 조회하거나 입력,수정,삭제할 수 있다.
- 이 어플리케이션의 핵심 기능은 과거의 연도들을  평가할 수 있다는데 있다.  평가는 총 7단계로 구성되며,  각각 다음과 같다.  1. 최고의 해, 2. 아주 좋았던 해, 3. 좋았던 해, 4. 보통의 해, 5. 나쁜 해, 6. 아주 나쁜 해 7. 최악의 해  
- 이 평가 점수는 사용자가 연도별로 선택할 수 있어야 하며, 연도 아이템에 표시될 때는 색상으로 표시된다.   최고의 해는 진한 녹색이며, 점수가 낮아질수록 컬러가 옅어진다. 최악의 해는 진한 붉은 색이며, 점수가 낮아질 수록 컬러가 옆어진다.  4번 보통의 해는 힌색이다.
- 그리고 현재까지의 평균 점수를 칼렌더 상단에 표시한다.  해당 점수에 맞는 컬러와 함께.
- 미래는 무조건 힌색이다. 점수를 줄 수 없으며, 예상되는 이벤트나 목표를 적을 수 있다.

* 데이터베이스
- 사용자의 데이터는 로컬에 json 으로 저장하며, 별도의 데이터베이스는 사용하지 않는다.  

* 언어
- 한국어가 기본 언어이며, 영어, 일본어를 추가로 지원한다. 설정에서 언어를 선택할 수 있어야 한다.
